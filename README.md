# 恒驰泵业财务系统

## 系统概述

恒驰泵业财务系统是一个基于Flask框架开发的企业财务管理系统，主要用于企业的日常财务收支管理、报表生成和数据分析。

## 核心功能

### 1. 交易管理
- 支持日常收支交易的录入、编辑和删除
- 交易分类管理（入账/出账）
- 交易明细分类支持可输入可选择
- 支持交易备注和搜索功能

### 2. 报表系统
- **日报表**：按日统计收支情况，包含上期结余、当日交易明细和当日汇总
- **月报表**：按月统计收支情况，包含上期结余、每日交易明细和月度汇总
- **年度报表**：按年统计收支情况，包含年初结余、月度交易汇总和年度汇总
- **PDF导出**：支持所有报表的PDF格式导出，包含上期结余信息

### 3. 数据分析
- 财务统计：显示总收入、总支出和结余
- 按年份、月份筛选交易数据
- 支持交易数据的CSV导出

### 4. 系统管理
- 用户登录和权限控制
- 数据备份和恢复
- 系统设置

## 技术栈

- **后端**：Python 3.9+, Flask, SQLAlchemy, SQLite
- **前端**：HTML5, CSS3, JavaScript, Bootstrap 5
- **PDF生成**：ReportLab
- **认证**：Flask-Login, Werkzeug

## 安装部署

### 环境要求
- Python 3.9 或更高版本
- pip 包管理器

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd 账务系统
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **初始化数据库**
   ```bash
   python app.py
   ```
   系统会自动创建数据库和表结构

4. **创建管理员用户**
   首次运行时，系统会自动创建默认管理员用户：
   - 用户名：admin
   - 密码：admin123

5. **启动服务**
   ```bash
   python app.py
   ```
   服务默认运行在 http://127.0.0.1:5000

## 系统使用

### 登录系统
1. 打开浏览器访问 http://127.0.0.1:5000
2. 输入用户名和密码登录

### 功能模块

#### 1. 首页
- 财务统计概览
- 最近交易记录
- 交易筛选功能

#### 2. 数据录入
- 新增交易记录
- 批量录入交易
- 最近录入的交易记录

#### 3. 财务报表
- 日报表：选择日期生成当日报表
- 月报表：选择年份和月份生成月度报表
- 年度报表：选择年份生成年度报表
- 支持PDF导出

#### 4. 数据导出
- 导出交易数据为CSV格式

## 注意事项

1. **数据备份**：定期备份 `data/finance.db` 文件
2. **安全设置**：生产环境中应修改默认管理员密码
3. **性能优化**：对于大量交易数据，建议定期清理或归档历史数据
4. **浏览器兼容性**：推荐使用Chrome、Firefox等现代浏览器
