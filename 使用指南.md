## 系统功能

### 1. 首页仪表板
- 显示财务统计概览（收入、支出、提现、结余）
- 按年份和月份筛选数据
- 显示最近的交易记录

### 2. 总账管理
- 查看所有交易记录
- 按类型（出账/收入/提现）筛选
- 按年份和月份筛选
- 添加新的交易记录
- 分页浏览（每页20条）

### 3. 月度账单
- 选择特定月份查看详细账单
- 显示月度收入、支出、提现和结余
- 分别列出出账明细、收入明细和店铺提现明细

### 4. 借款管理
- 查看所有借款记录
- 添加新的借款记录
- 标记借款为已归还
- 删除借款记录
- 显示借款统计（总借款、已归还、未归还）

### 5. 数据导入
- 支持从Excel文件导入财务数据
- 自动识别多种Sheet格式
- 显示导入结果统计

## 技术架构

### 后端
- **框架**: Flask 3.0.0

### 前端
- **技术**: 原生HTML/CSS/JavaScript
- **样式**: 响应式设计，支持移动端
- **交互**: AJAX异步请求

## 使用说明

### 启动系统

1. 确保已安装Python 3.8+
2. 安装依赖包:
   ```bash
   pip install -r requirements.txt
   ```

3. 初始化数据库（首次运行）:
   ```bash
   python database.py
   ```

4. 导入Excel数据（如需重新导入）:
   ```bash
   python import_data.py
   ```

5. 启动应用:
   ```bash
   python app.py
   ```

6. 在浏览器中访问: http://localhost:5000

### 日常使用

1. **查看财务概览**: 访问首页查看整体财务状况
2. **管理交易记录**: 进入"总账"页面查看和管理所有交易
3. **查看月度账单**: 进入"月度账单"页面查看特定月份的详细账单
4. **管理借款**: 进入"借款管理"页面管理借款记录
5. **导入新数据**: 进入"数据导入"页面上传新的Excel文件

### 数据筛选

在首页和总账页面，可以使用以下筛选条件:
- **年份**: 选择特定年份
- **月份**: 选择特定月份
- **类型**: 选择交易类型（出账/收入/提现）

### 添加交易记录

1. 进入"总账"页面
2. 点击"添加交易"按钮
3. 填写交易信息（日期、类型、类别、金额、描述）
4. 点击"保存"按钮

### 管理借款

1. 进入"借款管理"页面
2. 查看所有借款记录
3. 点击"归还"按钮标记借款为已归还
4. 点击"删除"按钮删除借款记录
5. 点击"添加借款"按钮添加新的借款记录

## 数据库设计

### 交易记录表 (transactions)
- id: 主键
- date: 交易日期
- type: 交易类型（出账/收入/提现）
- category: 交易类别
- amount: 交易金额
- description: 交易描述
- month: 所属月份
- year: 所属年份
- created_at: 创建时间

### 借款表 (loans)
- id: 主键
- date: 借款日期
- lender: 放款人
- amount: 借款金额
- status: 归还状态（已归还/未归还）
- return_date: 归还日期
- created_at: 创建时间

### 分配表 (distributions)
- id: 主键
- date: 分配日期
- person: 分配人员
- amount: 分配金额
- created_at: 创建时间

## API接口

### 交易记录
- GET /api/transactions - 获取交易列表
- POST /api/transactions - 创建交易记录
- PUT /api/transactions/<id> - 更新交易记录
- DELETE /api/transactions/<id> - 删除交易记录

### 借款管理
- GET /api/loans - 获取借款列表
- POST /api/loans - 创建借款记录
- PUT /api/loans/<id> - 更新借款记录
- DELETE /api/loans/<id> - 删除借款记录

### 统计数据
- GET /api/statistics - 获取统计数据

### 数据导入
- POST /api/import - 导入Excel数据
